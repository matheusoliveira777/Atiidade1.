RGB e CMYK

RGB (Red, Green, Blue)

O RGB é um modelo de cores aditivo usado em displays eletrônicos, como monitores, TVs e câmeras digitais. Ele combina as cores vermelho (Red), verde (Green) e azul (Blue) em diferentes intensidades para criar uma ampla gama de cores. Quando todas as cores são combinadas na máxima intensidade, o resultado é a cor branca.

CMYK (Cyan, Magenta, Yellow, Key/Black)

O CMYK é um modelo de cores subtrativo usado principalmente em impressão. Ele combina as cores ciano (Cyan), magenta (Magenta), amarelo (Yellow) e preto (Key/Black) para criar outras cores. Diferente do RGB, o CMYK começa com a luz branca e subtrai as cores para criar o espectro desejado. Quando todas as cores são combinadas, o resultado é uma cor próxima ao preto.
